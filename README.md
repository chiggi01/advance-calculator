# Python Calculator GUI

A simple and interactive Calculator application built using Python and Tkinter. This project provides a graphical user interface (GUI) for performing basic mathematical calculations and demonstrates the fundamentals of GUI development, event handling, and application logic in Python.

## 📌 Project Overview

This calculator was developed as a learning project to understand how desktop applications work using Python's Tkinter library. The application allows users to perform various arithmetic operations through an easy-to-use graphical interface.

## 🚀 Features

* Addition, Subtraction, Multiplication, and Division
* Percentage Calculation
* Square Root Function
* Positive/Negative Number Toggle (+/-)
* Clear (AC) Functionality
* User-Friendly Interface
* Responsive Button Layout
* Error Handling for Invalid Inputs

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI Library)
